# 🐶 Homes in Love

**Homes in Love** هو موقع عربي حديث لعشاق الكلاب يقدم تجربة كاملة وممتعة حول عالم الكلاب من خلال صفحات تعريفية وخدمية وبصرية منظمة تشمل السلالات والرعاية والخدمات والمعرض والتواصل

الموقع مبني بواجهة أمامية احترافية باستخدام HTML و CSS و JavaScript مع تصميم عربي باتجاه RTL وتجربة استخدام متجاوبة تدعم الهاتف والكمبيوتر

---

## 📌 نبذة عن المشروع

فكرة المشروع هي إنشاء موقع بسيط وراقي يساعد المستخدم على التعرف على الكلاب بطريقة جميلة وسهلة من خلال محتوى عربي واضح وتصميم بصري قوي يجمع بين اللون الأخضر الداكن والذهبي مع مؤثرات حركة وخلفيات تفاعلية وأدوات وصول تساعد المستخدمين المختلفين

الموقع مناسب كمشروع واجهة أمامية Frontend ويمكن تطويره لاحقا ليصبح منصة خدمات كاملة لأصحاب الكلاب مثل الحجز والاستشارات والتدريب والمتجر الإلكتروني

---

## 🌐 صفحات الموقع

| الصفحة | الملف | الوصف |
|---|---|---|
| الرئيسية | `index.html` | الصفحة الأساسية للموقع وتعرض هوية Homes in Love ونظرة عامة جذابة |
| عن الكلاب | `about.html` | صفحة تعريفية تقدم معلومات عامة عن الكلاب وقيمتها في حياة الإنسان |
| السلالات | `breeds.html` | صفحة تعرض سلالات الكلاب ومعلومات تساعد المستخدم على معرفة الفروقات بينها |
| الرعاية | `care.html` | صفحة مخصصة لنصائح العناية والتغذية والنظافة والاهتمام اليومي |
| الخدمات | `services.html` | صفحة تعرض الخدمات التي يمكن تقديمها لأصحاب الكلاب |
| المعرض | `gallery.html` | صفحة بصرية لعرض صور الكلاب بطريقة جذابة |
| تواصل معنا | `contact.html` | صفحة للتواصل مع إدارة الموقع أو طلب خدمة |

## ✨ المميزات الرئيسية

### 🎨 تصميم حديث

- واجهة عربية كاملة باتجاه RTL
- ألوان فخمة تجمع بين الأخضر الداكن والذهبي
- خلفيات متحركة وطبقات ضوئية ناعمة
- بطاقات منظمة وتأثيرات Hover
- تصميم مناسب لموقع عن الحيوانات الأليفة والكلاب

### 🌙 أوضاع عرض متعددة

- الوضع الداكن Dark Mode
- الوضع الفاتح Light Mode
- وضع التباين العالي High Contrast
- دعم خط مناسب لعسر القراءة Dyslexic Font

### ♿ إمكانية الوصول

الموقع يحتوي على لوحة أدوات مساعدة للمستخدم تشمل خصائص مثل تغيير المظهر وتحسين القراءة وتسهيل التصفح مما يجعله أقرب لمعايير الوصول الحديثة

### 📱 تصميم متجاوب

- يعمل على شاشات الكمبيوتر
- مناسب للهواتف
- مناسب للأجهزة اللوحية
- قوائم وتنقلات قابلة للاستخدام على الشاشات الصغيرة

### 🖱 تجربة تفاعلية

- مؤشر مخصص Custom Cursor
- تأثيرات حركة Animation
- خلفيات Mesh و Orbs و Particles
- انتقالات ناعمة بين العناصر
- أزرار تفاعلية واضحة

---

## 🛠 التقنيات المستخدمة

- **HTML5** لبناء هيكل الصفحات
- **CSS3** للتصميم والحركات والتجاوب
- **JavaScript** للتفاعل والتحكم في الأدوات
- **Font Awesome** للأيقونات
- **Google Fonts** لاستخدام خط Tajawal وخط OpenDyslexic
